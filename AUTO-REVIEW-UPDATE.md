# 自动复习更新工作流

## 🗣️ 自然语言触发

### 触发词

当用户说以下任意一句话时，自动触发复习更新：

```
- 今天的错题复习完了
- 错题复习完了
- 复习完了
- 今天复习完了
- 物理错题复习完了
- 数学错题复习完了
- 英语错题复习完了
- 今天有什么要复习的
- 今天的复习提醒
```

---

## 🤖 奴才响应流程

### Step 1：识别意图

```
用户：今天的错题复习完了
     ↓
识别：复习更新意图
     ↓
提取：学生姓名（从上下文）、学科（可选）、掌握情况（可选）
```

### Step 2：执行更新

```bash
python3 skills/mistake-notebook/scripts/update-review.py \
  --student <学生姓名> \
  --today \
  --mastered good
```

### Step 3：回复用户

```
✅ 复习进度已更新！

今日完成复习：3 道
  • 数学：1 道（程序运算与不等式组）
  • 物理：2 道（牛顿第一定律 ×2）

下次复习日期：2026-04-03（3 天后）

继续保持，按时复习哦！💪
```

## 🔧 实现方式

### 方式 1：Agent 自动识别（推荐）

在 Agent 的 system prompt 中添加：

```
当用户说"复习完了"、"错题复习完了"等类似话语时，
自动调用 update-review.py 脚本更新复习进度。

执行命令：
python3 skills/mistake-notebook/scripts/update-review.py \
  --student <学生姓名> \
  --today \
  --mastered good
```

### 方式 2：消息触发器

配置消息触发器，当收到包含特定关键词的消息时，自动执行脚本。

---

## 💡 最佳实践

1. **自动识别**：用户无需知道命令，直接说自然语言
2. **智能反馈**：更新后显示详细统计
3. **鼓励机制**：完成复习后给予鼓励
4. **下次提醒**：告知下次复习日期
