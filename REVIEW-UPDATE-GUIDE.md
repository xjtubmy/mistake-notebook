## 📊 参数说明

### --today（批量更新）

| 参数 | 说明 |
|------|------|
| `--today` | 更新今日到期的所有错题 |
| `--subject <学科>` | 按学科筛选（可选） |
| `--mastered <掌握情况>` | poor/fair/good/excellent（默认 good） |

### --id（单题更新）

| 参数 | 说明 |
|------|------|
| `--id <错题 ID>` | 错题 ID（如 20260331-001） |
| `--round <轮次>` | 复习轮次（0-5） |
| `--mastered <掌握情况>` | poor/fair/good/excellent（默认 good） |

### --mastered（掌握情况）

| 值 | 说明 | 下次复习建议 |
|----|------|-------------|
| `poor` | 很差 | 1-2 天后重新复习 |
| `fair` | 一般 | 按正常间隔 |
| `good` | 良好 | 按正常间隔（推荐） |
| `excellent` | 优秀 | 可考虑跳过下轮 |

## 🎯 完整复习流程

### Step 1：收到复习提醒

每天 18:00 收到飞书消息：

```
📅 **复习提醒**

**待复习**：3 道

**📚 物理**（2 道）
• 力的合成（第 1 轮）🟡 今日
• 牛顿第一定律（第 1 轮）🟡 今日

**📚 数学**（1 道）
• 程序运算与不等式组（第 1 轮）🟡 今日
```

### Step 2：完成复习

1. 打开错题本（PDF 或长图）
2. 盖住答案，独立重做
3. 对照解析，理解思路
4. 在复习记录表上打勾

### Step 3：告诉奴才复习完成了

**对奴才说**：
```
今天的错题复习完了
```

奴才会自动：
- ✅ 更新所有今日复习的错题进度
- ✅ 计算下次复习日期
- ✅ 记录掌握情况

### Step 4：等待下次复习提醒

2026-04-03 18:00，系统自动发送复习提醒。

## ⚠️ 常见问题

### Q1: 忘记告诉奴才怎么办？

**A**: 随时告诉奴才：
```
昨天的错题复习完了
```

奴才会自动补更新。

### Q2: 复习后还是不会怎么办？

**A**: 告诉奴才：
```
今天的错题复习完了，但是掌握得不好
```

奴才会：
- 标记为掌握差
- 建议重新学习相关概念
- 安排 1-2 天后重新复习

### Q3: 可以一次性更新多天的错题吗？

**A**: 可以，告诉奴才：
```
过去 3 天的错题都复习完了
```
